HOW TO SET OR RESET THE TEACHER PASSWORD (UKEPLAN)
===================================================

The teacher password is stored as a SHA-256 hash in the Apps Script's
Script Properties. It is never written in any source file. This is the
NEW Ukeplan backend — separate from the vurderingskalender script and
its own separate password.

To set or change the password:

1. Open the Google Sheet that holds the "Planelementer" sheet
2. Go to Extensions → Apps Script
3. Add the following function temporarily at the bottom of the script:

---

function _setPassword() {
  setupPassword("skriv-passordet-her");
}

---

4. Replace "skriv-passordet-her" with the new password.
5. Select _setPassword from the function dropdown and click Run.
6. Check the Execution log — it should say:
   "Password hash stored successfully."
7. Delete the _setPassword function from the script (so the plaintext
   password is not left in the editor).

The setupPassword function itself (already in the deployed script):

---

function setupPassword(plaintext) {
  PropertiesService.getScriptProperties()
    .setProperty('PASSWORD_HASH', hashString(plaintext));
  Logger.log('Password hash stored successfully.');
}

---

NOTE: You do not need to redeploy the web app after changing the password.
Script Properties are read at runtime, so the new hash takes effect
immediately for all subsequent login attempts.
